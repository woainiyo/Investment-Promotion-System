🎉 静态资源托管成功！

项目名称：Beshe 后端系统

静态资源路径：/static/

访问方式：http://localhost:3000/static/index.html

这个页面通过 Express 的静态资源托管功能提供服务。

您可以在 static 目录下放置任何静态文件，如：

  • HTML 文件
  • CSS 样式表
  • JavaScript 文件
  • 图片 (PNG, JPG, GIF 等)
  • 字体文件

✅ CSS 和 JavaScript 测试

如果看到这个绿色背景的容器，说明CSS文件加载成功！

注意：静态资源不需要 JWT 认证，可以被任意用户访问。
